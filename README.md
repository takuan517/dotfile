# dotfiles

&emsp;
## Overview

This repository contains scripts to set macOS.

&emsp;

## Required

- [macOS](https://www.apple.com/jp/macos/big-sur/) - v11.4

&emsp;

## Install

Download installation materials.

```shell
$ cd ~ && git clone https://github.com/takuan517/dotfiles
```

Set macOS.

```shell
$ cd ~/dotfiles && make
```
